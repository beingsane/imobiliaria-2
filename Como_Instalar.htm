Instalando o e-Mobiliaria

1º passo: Entre no cpanel do seu servidor e crie um banco de dados mysql.

2º passo: Abra o arquivo "configuracao_mysql.php" e altere as variáveis como abaixo:

//Nome do host do seu banco de dados
$db_host = "localhost";

//Nome de usuário do banco de dados
$db_username = "usuario";

//Senha do usuário do banco de dados
$db_password = "senha";


//Nome do banco de dados que você criou
$db_name = "nome";


//Informe o endereço (url) onde estão os arquivos do site
$site_url = "http://www.emobiliaria.com";

3º passo: Envie todos os arquivos do site para seu servidor, via FTP.

4º passo: Altere as permissões destes diretórios:

"banners" ----- -> chmod 777
"fotos_anuncios" -------> chmod 777


5º passo: Instale o banco de dados acessando este endereço: http://www.seudominio.com/instalar.php

6º passo: Após a instalação é MUITO IMPORTANTE que você exclua o arquivo "instalar.php" do seu servidor para evitar que alguém o execute.

7º passo: Acesse o painel de administração emt: http://www.seudominio.com/admin
Nome: admin
Senha: admin

8º passo: É essencial que agora você acesse a administração e clique no menu "Alterar Senha" para que você escolha uma senha segura.
